Sponsor Logo Sizes

One canonical number per firm. The same firm gets the same number on every event page so logos always match across NY, Dubai and European. Numbers are desktop max-heights in px, logos never stretch.

How sizing works

Every event code block has a LOGO SIZES, EDIT HERE section at the very top. One number per logo. Change it there and it updates everywhere in that block. If the firm is on more than one page, change it in both blocks and update this page.

--logo-scale (currently 0.8) scales every logo at once. If everything feels big or small, this is the one number to change. Keep it identical in every event file so shared firms stay matched.

--mobile-scale (currently 0.75) sets mobile size automatically as a pure proportional scale (no floor), so all logos resize together and there are no separate mobile numbers to maintain. Rows wrap on mobile so logos stay readable, and dividers that land on a line break hide themselves.

Rendered desktop size = number × 0.8. Mobile = number × 0.8 × 0.75. If everything reads too small on a phone, raise --mobile-scale.

Row spacing rules

  • Normal tier rows: logos spread the card width, dividers count as items, every gap identical.
  • Split-label cards (2 or 3 eyebrow labels): each logo centres in its half or third, the divider pins exactly to the label boundary.
  • Sparse single-tier rows (1 to 3 logos, like Dubai Gold): packed from the left with a fixed 64px gap, not stretched to the card edges.
  • Mobile: rows wrap and pack from the left with a fixed gap, like text wrapping, never stretched to fill the row; only split-label cards keep their logos centred within their halves or thirds. Stray dividers at line breaks hide automatically.

When a firm joins a new event, look it up here first and reuse its number. New firms get sized once and added to this page.
